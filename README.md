# Daydream

Мой первый discord-bot (тестируемый)

## Установка
1. клонируйешь репозиторий: `git clone https://github.com/Foxarion/daydream`
2. установливаешь зависимости: `pip install -r requirements.txt`
3. настраиваешь `.env` / Добавляешь токен бота в виде TOKEN=... 
4. запускаешь бота используя команду в терминале: `python main.py`

## Библиотеки и язык
- Python 3.x
- disnake
- python-dotenv
